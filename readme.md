# Car Service Booking - PostgreSQL version

## Setup

```bash
npm install
```

Create PostgreSQL database:

```sql
CREATE DATABASE car_service_db;
```



Run:

```bash
npm run dev
npm run restart
```

Default admin:

```txt
email: 
password: 
```

Main pages:
- `/register`
- `/login`
- `/bookings/create`
- `/bookings`
- `/admin`
